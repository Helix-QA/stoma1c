Документ "Операция по бонусному счету"

Документ предназначен для проведения операций по начислению либо списанию бонусов на бонусном счете.

В документе необходимо указать значения следующих полей:

  • Операция - вид операции документа. Операция может быть начислением либо списанием.
  • Сумма - сумма операции.
  • Основное. Группа реквизитов:
    • Владелец. Переключатель типа владельца. Может принимать значение карта либо пациент.
    • Бонусный счет. Поле выбора бонусного счета.
    • Содержание. Указывается содержание операции.

  • Дополнительно. Группа реквизитов:

    • Номер. Номер документа.

    • Дата. Дата документа.

    • Клиника. Указывается клиника из справочника клиник.

    • Комментарий. Произвольный комментарий к документу.
