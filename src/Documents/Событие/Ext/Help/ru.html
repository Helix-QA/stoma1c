Документ "Предварительная запись"

Документ предназначен для регистрации запланированных приемов пациентов. Иными словами, создать предварительную запись можно тогда, когда необходимо записать пациента на предполагаемое им посещение клиники.

Ввод данного документа, а иными словами оформление предварительной записи, осуществляется в Журнале записи на Рабочем столе.

При оформлении предварительной записи необходимо заполнить следующие поля:

  • Статус. В данном поле указывается текущий статус предварительной записи. Статусы могут быть:
    • Не подтверждено;
    • Запланировано;
    • Пациент ожидает;
    • Пациент в кресле;
    • Прием окончен;
    • Прием не состоялся;
    • Прием отменен;
  • Дата. Определяет день на который данная предварительная запись будет оформлена. В данном поле может быть указан день, совпадающий с сегодняшним днем календаря, либо какой-либо предстоящий день. Дата события автоматически устанавливается программой исходя из того в области какого дня Журнала предварительной записи эта предварительная запись была оформлена.
  • Время начала. Значение данного поля указывает время начала записи. Значение данного поля, также как и значение в поле "Дата" автоматически устанавливается программой на основании области Журнала предварительной записи в которой данный документ был оформлен.
  • Время окончания. Значение в данном поле указывает на время окончания записи. Первоначально, когда список услуг в документе не заполнен, время окончания события автоматически устанавливается на 15 минут большим, чем время начала. При заполнении списка услуг в документе, время окончания события рассчитывается исходя из общей длительности всех услуг документа. Которая, в свою очередь, рассчитывается на основании нормы времени, указанной для каждой услуги (см. справочник "Номенклатура").
  • Длительность. Длительность предварительной записи в минутах.
  • Пациент. В данном поле необходимо указать пациента.
  • Причина записи. В данном поле указывается причина предварительной записи. Выбирается из справочника "Причины записи".
  • Врач. В данном поле указывается принимающий врач. Выбирается из справочника "Сотрудники".
  • Кабинет. В данном поле указывается кабинет приема. Выбирается из справочника "Кабинеты".
  • Услуги. Табличная часть, в которой указываются те услуги, которые будут оказаны клиенту. В данную табличную часть могут быть добавлены только услуги. Для удобства заполнения списка услуг можно воспользоваться подбором номенклатуры
    • Услуга. В данное поле вводится элемент справочника номенклатуры.
    • Цена - цена оказываемой услуги.
    • Мин.- норма времени, т.е. среднее время, затрачиваемое на оказание данной услуги клиенту.
  • Комментарий. Произвольный комментарий к документу.
  • Причина отмены - в случае, если оформленную предварительную запись необходимо отменить, в данном поле необходимо указать причину отмены предварительной записи. Значение выбирается из справочника "Причины отмены записи".
  • Метки. В данном поле указыываются основные важные метки по записи: страховая, острая боль, первичный прием, повторный прием.

Если клиент пришел согласно предварительной записи в назначенное время, тогда на основании данной предварительной записи необходимо оформить документ "Прием".
