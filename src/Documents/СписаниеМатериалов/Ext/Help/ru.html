Документ "Списание материалов"

Документ предназначен для оформления факта списания со склада расходных материалов (материалов списания), т.е. тех материалов, которые были израсходованы для оказания услуг пациенту. Переход к списку документов производится из раздела "Запасы" / Группа "Материалы" / "Списания материалов".

Важно. Данный документ предназначен для списания только расходных материалов (материалов списания). Для того чтобы оформить списание со склада товаров, например, по результатам проведенной инвентаризации, предназначен документ "Списание запасов".

 

В документе необходимо заполнить значения следующих полей:

  • Вид операции. Указывается вид операции - по складу или сотруднику.
  • Склад. В этом поле необходимо указать склад, из которого будут списаны расходные материалы. Значение склада в этом поле является значением "по  умолчанию". Т.е. если в табличной части "Запасы" для какого-либо расходного материала не будет указан склад списания, значение склада будет взято из этого поля.  
  • Персонал. В этом поле необходимо указать сотрудника, из которого будут списаны расходные материалы. Значение склада в этом поле является значением "по  умолчанию". Т.е. если в табличной части "Запасы" для какого-либо расходного материала не будет указан сотрудник списания, значение сотрудника будет взято из этого поля. 
  • Запасы. Табличная часть:
    • Номенклатура. В данное поле вводится элемент справочника номенклатуры.
    • Характеристика, серия. Указывается характеристика номенклатурной позиции. Поле доступно только в том случае, если при настройке параметров учета указано ведение учета по партиям номенклатуры. Выбирается из справочника "Характеристики, серии".
    • Количество. Указывается количество единиц измерения номенклатурной позиции.
    • Упак. упаковка номенклатуры, если есть. Выбирается из справочника "Упаковки номенклатуры".
    • Склад. В этом поле необходимо указать склад, из которого будут списаны расходные материалы данной строки.
    • Документ продажи. В данном поле необходимо указать один из документов "Отчет о розничных продажах", который является основанием для списания материалов.
    • Пациент. Указывается пациент, которому была оказана услуга, оказание которой потребовало расхода данного материала списания. Выбирается из справочника "Контрагенты".
    • Сотрудник. Выбирается сотрудник, оказавший услугу, потребовавшую расхода данного материала списания. Выбирается из справочника "Сотрудники".
    • Услуга. Указывается услуга, оказание которой потребовало расхода данного материала списания. Выбирается из справочника "Номенклатура".
    • Структурная единица. Выбирается значение справочника "Структурные единицы".
  • Комментарий. Произвольный комментарий к документу.

 

При проведении документа, указанные в нем расходные материалы будут фактически списаны со склада в соответствии с указанными настройками складского учета. (Настройки складского учета могут быть изменены на форме настройки параметров учета).
