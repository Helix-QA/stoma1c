Печать этикеток и ценников

Обработке предназначена для печати этикеток и ценников по разработанным шаблонам.

В области Цены указываются параметры для расчета цен. Здесь необходимо указать структурную единицу, для которой необходимо рассчитывать цены, вид цен и дату на которую должны быть рассчитаны цены.

В области Параметры печати необходимо выбрать шаблон для печати и указать тот принтер, на котором должна быть произведена печать.

В области Отбор нужно указать критерии, по которым программа отберет номенклатуру для печати этикеток или ценников из общего списка номенклатуры. Если не указать никаких параметров отбора, тогда в таблицу товаров попадут все элементы справочника Номенклатура.

Наконец, таблица Товары содержит все элементы номенклатуры, для которых будет производиться печать этикеток или ценников.
                
Для того чтобы заполнить таблицу товаров необходимо выбрать один из вариантов заполнения из раскрывающегося меню Заполнить.
Если выбрать вариант Заполнить по остаткам то будут выбраны те товары, которые есть на складе. При выборе варианта Заполнить по номенклатуре таблица товаров будет заполнена согласно установленным отборам.

Флажок в поле Выбран указывает на то, будет ли для данного элемента из таблицы товаров осуществляться печать, а значение в колонке На печать показывает то, сколько именно экземпляров будет напечатано.

При помощи разворачивающегося меню Количество для печати можно установить количество экземпляров сразу для всех строк таблицы.
