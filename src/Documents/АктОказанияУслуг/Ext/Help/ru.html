Документ "Акт оказания услуг"

Документ предназначен для формирования акта об оказании услуг и выставления счета организации-страховщику за оказанные пациентам услуги.

Переход к списку актов оказания услуг производится из раздела "Оказание услуг" / Группа "Страховые компании" / "Акты оказанных услуг".

Необходимо заполнить следующие реквизиты:

  • Контрагент - страховую компанию, для которой формируется акт
  • За период с… по…  период для отбора.

 

Документ имеет табличную часть Документы основания. В таблице Документы основания указываются приемы, которые были проведены на основании действующего полиса выбранной страховой компании. При помощи разворачивающегося меню Заполнить приемами можно автоматически заполнить таблицу приемами, проведенными по полису выбранной компании. Также предусмотрено ручное добавление приемов в таблицу.

Документы основания. Табличная часть:

  • Документ основания - документ Прием.
  • Сумма - сумма документа основания.
