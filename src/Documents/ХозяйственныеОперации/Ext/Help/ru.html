Документ "Ручная операция"

Документ предназначен для создания произвольной проводки по управленческому плану счетов. Переход к списку документов производится из раздела "Финансы" / Группа "Хозяйственные операции" / "Ручная операция".

В документе необходимо заполнить значения следующих полей:

  • Структурная единица. В этом поле необходимо указать структурную единицу организации, в которой заключается договор с клиентом.
  • Проводки. Табличная часть:
    • Счет ДТ. Счет дебета.
    • Субконто ДТ1. Субконто дебета.
    • Субконто ДТ2. Субконто дебета.
    • Счет КТ. Счет кредита.
    • Субконто КТ1. Субконто кредита.
    • Субконто КТ2. Субконто кредита.
    • Количество. Количество операции.
    • Сумма. Сумма операции.
    • Содержание. В этом поле указывается содержание проводки.
  • Комментарий. В данное поле вводится комментарий к документу.
