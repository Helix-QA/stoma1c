ЗАГРУЗКА ДАННЫХ ИЗ ВНЕШНЕГО ИСТОЧНИКА

 

Какие данные можно загружать и куда?

 

Поддерживается импорт из форматов xlsx, mxl, csv. Загружать можно в:

  • Табличные части документов Заказ поставщику, Заказ покупателю, Ввод начальных остатков;
  • Справочники Номенклатура, Контрагенты, Сотрудники;
  • Регистр сведений Цены номенклатуры;

ВНИМАНИЕ! Операции загрузки данных в справочники и регистр сведений являются необратимыми! Т.е. в случае загрузки ошибочных данных (неактуальной информации) справочник, или регистр сведений, придется вычищать самостоятельно.

 

 

Как воспользоваться загрузкой?

 

Команды начала работы с механизмами доступны пользователям непосредственно из объектов. Так, например, что бы загрузить данные в табличную часть достаточно открыть соответствующий документ, а для загрузки элементов справочников вам необходимо перейти в форму списка.

 

Ссылки для быстрого перехода:

  •  Справочник "Контрагенты"
  •  Справочник "Номенклатура"

ВНИМАНИЕ! Загрузка данных доступна только пользователям с правами администратора приложения.

 

Типы данных.


Что бы успешно сопоставить ссылочные данные  (например: "Номенклатура, Клиент, Сотрудник, № партии и др."), в файле следует указать название в виде строки.

Данные типа "Булево", на формах обычно выглядит в виде флажка или переключателя. Можно указать в виде (+, -, ИСТИНА, ЛОЖЬ, Да, Нет, 0 , 1).

Дату необходимо указывать в виде строки 02.03.1999 или 02/03/1999.

Остальные типы можно передавать в виде строкового наименования.


Бывают случаи, когда некоторые элементы по типу партии номенклатуры, могут быть по несколько штук у одной номенклатуры. Для этого нужно продублировать ключевые поля для поиска номенклатуры и для каждой строки указать нужную партию. Получится что несколько строк с одинаковой номенклатурой, но разными партиями. Это касается и других аналогичных элементов.
