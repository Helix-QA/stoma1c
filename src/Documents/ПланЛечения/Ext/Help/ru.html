Документ "План лечения"

Документ "Плн лечения" предназначен для составления плана лечения пациента. В ходе лечения может быть составлено несколько планов лечения. Принятый пациентом план лечения указывается статусом Принят. Переход к списку документов производится из раздела "Оказание услуг" / Раздел "Оказание услуг" / "Планы лечения". 

В документе необходимо указать значения следующих полей:

  • Пациент. В этом поле необходимо указать пациента, которому будет проведено лечение. Значение выбирается из справочника "Клиенты".
  • Страховой полис. В этом поле необходимо указать страховой полис, по которому будут производиться взаиморасчеты.
  • Врач. В этом поле необходимо указать сотрудника организации, который будет осуществлять лечение.
  • Статус. В этом поле необходимо указать статус плана лечения.

 

  • Лечение. Табличная часть:
    • Номер приема. В данном поле указывается порядковый номер приема.
    • Зуб. В данное поле вводится элемент перечисления Зубы.
    • Услуга. В данное поле вводится элемент справочника номенклатуры.
    • Кол-во. Указывается количество единиц измерения номенклатурной позиции.
    • Цена - цена реализуемой номенклатурной позиции.
    • % ручн. скидки - % ручной скидки по строке.
    • Сумма авт. скидки - сумма автоматической скидки по данной строке.
    • Сумма - стоимость услуги по данной строке.
    • Прием/Наряд-заказ. В данном поле вводится документ основания - Прием либо Наряд-Заказ.

 

  • Скидки и наценки. Табличная часть. Указаны все автоматически рассчитанные скидки по данному Приему. 
  • Дополнительно. Табличная часть. Содержит служебные поля. 

 

На основании данного документа могут быть оформлены следующие документы:

  • "Наряд-заказ";
  • "Прием";
