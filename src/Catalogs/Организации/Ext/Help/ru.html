Справочник "Организации"

Справочник предназначен для ведения списка собственных организаций, входящих в состав компании, и хранения постоянных сведений о них.

 

Редактирование информации об организации производится в отдельном диалоговом окне, открываемом из формы списка справочника. При этом указывается информация, характеризующая организацию:

  • Префикс. Указывается префикс, который присваивается при нумерации документов данной организации.
  • Юр. / физ. лицо. Признак, является ли организация юридическим или физическим лицом. 
  • Полное наименование. Указывается полное официальное наименование организации  для вывода в печатные формы документов.
  • Общие. Закладка:
    • Коды. Группа реквизитов:
      • ИНН - идентификационный номер налогоплательщика. Цифровой код, упорядочивающий учет налогоплательщиков в РФ.
      • КПП - код причины постановки на учет. Девятизначный цифровой код.
      • ОКПО - общероссийский классификатор предприятий и организаций.
    • Значения по умолчанию. Группа реквизитов:
      • Банковский счет организации по умолчанию. Используется для подстановки в документы. Выбирается из справочника "Банковские счета".
      • Система налогообложения. Доступны следующие варианты: Упрощенная система налогообложения, Единый налог на вмененный доход, Общая система налогообложения.
  • Адреса, телефоны. Закладка:
    • E-mail организации.
    • Другое. Указывается любая другая контактная информация.
    • Почтовый адрес организации.
    • Телефон организации.
    • Факс организации.
    • Фактический адрес организации.
    • Юридический адрес организации.

 

Кроме перечисленных выше параметров, для каждой организации можно определить перечень имеющихся банковских счетов.
