Отчет "Отчет по SMS-рассылкам"

Данный отчет предоставляет данные о текущем статусе SMS-рассылок, в частности о том, каким клиентам сообщения уже доставлены, а каким еще нет.

Настройка отчета производится посредством выбора одного из вариантов отчета, либо при нажатии кнопки "Настройка".

Варианты отчета

Выбор варианта отчета возможен путем нажатия кнопки "Выбор варианта". Текущий вариант отчета отображается в поле "Вариант отчета".

Отборы

Для отчета доступны следующие отборы:

  • По периоду. В отчете будут представлены данные по тем рассылкам, которые были осуществлены в течение указанного периода.
  • По виду рассылки. Отчет представит данные по тем SMS-рассылкам, которые соответствуют заданному виду рассылки.
  • По клиенту. Будут представлены данные о статусе сообщений, отправленных выбранному клиенту.
  • По статусу сообщений. В отчете будут представлены данные по SMS-сообщениям выбранного статуса. Например, можно получить список всех сообщений, которые до сих пор не доставлены клиентам.
