Документ "Оприходование запасов"

Документ предназначен для оформления факта оприходования материальных ценностей на склад либо подотчет сотрудника. Переход к списку документов производится из раздела "Запасы" / Группа "Инвентаризация" / "Оприходования запасов".

Документ может быть оформлен на основании проведенной инвентаризации или как свободный документ в случае произвольного оприходования запасов.

Необходимо заполнить следующие реквизиты:

  • Вид операции. В зависимости от вида операции документ может оприходовать запасы на склад, на сотрудника либо служить документом ввода начальных остатков.
  • Основание. Выбор или просмотр документа, инвентаризации, на основании которого производится оприходование.
  • Склад. В этом поле необходимо указать склад, в который будут оприходованы запасы.
  • Статья доходов. В этом поле указывается статья доходов при оприходовании запасов.
  • Вид цен. В этом поле необходимо указать вид цен, по которому будут оприходованы запасы.
  • Запасы. Табличная часть:
    • Номенклатура. В данное поле вводится элемент справочника номенклатуры.
    • Характеристика, серия. Указывается характеристика номенклатурной позиции. Поле доступно только в том случае, если при настройке параметров учета указано ведение учета по партиям номенклатуры. Выбирается из справочника "Характеристики, серии".
    • Количество. Указывается количество единиц измерения номенклатурной позиции.
    • Упак. - упаковка номенклатурной позиции.
    • Цена. Цена, по которой приходуется запас.
    • Сумма. Сумма, на которую приходуется запас.
    • Статья доходов. В этом поле указывается статья доходов при оприходовании запасов по номенклатурной позиции.
  • Комментарий. Произвольный комментарий к документу.

 

Данный документ может быть оформлен на основании документа "Инвентаризация запасов".
