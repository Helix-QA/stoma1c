Справочник "Структурные единицы"

Справочник "Структурные единицы" содержит список подразделений предприятия. При оформлении документов указание подразделения предприятия является обязательным. В программе предусмотрена возможность анализа выручки и себестоимости продаж раздельно по каждому подразделению торгового предприятия.

В справочнике необходимо указать значения для следующих полей:

  • Наименование. Указывается произвольное текстовое наименование.
  • Организация. Указывается та организация, частью которой является данное подразделение.
  • Основная касса. Из справочника "Кассы" выбирается та касса, которая будет считаться основной для данного подразделения.  
  • Основной склад. Выбирается тот склад, который будет являться основным для данного подразделения.
  • Вид цен реализации. Для организации выбирается собственный вид цен реализации. Выбирается из справочника "Виды цен".
  • Расписание работы. Представляет собой табличное поле в котором флажками напротив дня недели можно отметить рабочие дни данного подразделения, а также указать часы работы в каждый конкретный день недели.
