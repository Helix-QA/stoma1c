Справочник "Виды взаимодействий"

Предназначен для создания и определения вида задачи. По какому поводу была создана задача.

Для вида взаимодействия необходимо заполнить следующие реквизиты:

  • Наименование - краткое наименование вида взаимодействия.
  • Тема по умолчанию - текст в этом поле, будет использоваться для подстановки в соответствующие поля из других форм.
  • Способ взаимодействия - набор стандартных взаимодействий.
  • Картинка - используется для быстрого визуального восприятия.
