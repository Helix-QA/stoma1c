Отчет "Взаиморасчеты"

Отчет "Взаиморасчеты" предоставляет данные касающиеся взаиморасчетов с клиентами.

Варианты отчета

Выбор варианта отчета возможен путем нажатия кнопки "Выбор варианта". Текущий вариант отчета отображается в поле "Вариант отчета".

Для данного отчета доступны следующие варианты:

  • Взаиморасчеты с контрагентами. Предоставляются данные по взаиморасчетам организации с поставщиками, банками и т.д. Другими словами, в данном варианте предоставляются данные по взаиморасчетам со всем контрагентами, которые не являются клиентами организации.
  • Взаиморасчеты с клиентами. Предоставляются данные по взаиморасчетам с клиентами.
