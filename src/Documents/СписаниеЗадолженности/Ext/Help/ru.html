Документ "Списание задолженности"

Документ "Списание задолженности" предназначен для списания задолженности пациента перед клиникой. Переход к списку документов производится из раздела "Финансы" / Раздел "Взаиморасчеты" / "Списание задолженности". 

В документе необходимо указать значения следующих полей:

  • Пациент. В этом поле необходимо указать пациента, которому будет списана задолженность. Значение выбирается из справочника "Контрагенты".
  • Статья расходов. В этом поле необходимо указать статью расходов для данного документа.
  • Сумма списания. В этом поле необходимо указать сумму списания.

Документ имеет табличную часть на которой располагается расшифровка по документам основания.

  • Расшифровка. Табличная часть:
    • Документ основания. Указывается документ основания.
    • Документ аванса. Указывается документ аванса.
    • Пояснение - пояснение чей долг.
    • Сумма документа - сумма документа основания.

    • Сумма списания - сумма списания задолженности.

    • Статья расходов. В этом поле необходимо указать статью расходов для данного документа.
       
  • Комментарий - поле ввода, предназначенное для указания любой дополнительной информации к документу.
