Справочник "Упаковки номенклатуры"

Справочник предназначен для хранения перечня упаковок товара. Упаковка может использоваться в качестве дополнительной единицы измерения, и связана с базовой единицей определенным коэффициентом пересчета.  

Ввод новой упаковки

Для новой упаковки следует заполнить следующие поля:

  • Наименование. Наименование упаковки номенклатуры.
  • Номенклатура. Выбирается номенклатура, которая будет владельцем данной упаковки.
  • Коэффициент. Количество единиц в одной упаковке. Указывается по отношению к той базовой единице измерения, которая выбрана для позиции номенклатуры.
