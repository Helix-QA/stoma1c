Отчет "План-фактный анализ отработанного времени"

Отчет "План-фактный анализ отработанного времени" предоставляет данные времени, которые сотрудники должны были отработать по плану и о времени, которое было фактически отработано.

Настройка отчета производится посредством выбора одного из вариантов отчета, либо при нажатии кнопки "Настройка".

Варианты отчета

Выбор варианта отчета возможен путем нажатия кнопки "Выбор варианта". Текущий вариант отчета отображается в поле "Вариант отчета".

Для данного отчета доступны следующие варианты:

  • В виде таблицы. В данном варианте отчета данные о времени по плану и фактически отработанном времени выводятся в виде таблицы.
  • В виде диаграммы. Как и следует из названия, данные, в этом случае, выводятся графически, в виде диаграммы.
