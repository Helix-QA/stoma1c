Справочник "Виды сделок"

Справочник служит для указания всех возможных видов сделок с пациентами. При создании новой сделки всегда будет выбор из видов сделок.

В данный справочник вносится следующая информация:

  • Доступно - указывается ограничение по клиникам. Если ограничения нет, то необходимо выбрать "В любой клинике". Ограничение действует на отображение сделок с текущим видом в Мониторе (Продаж), при создании новой сделки, при изменении текущей.
  • Этапы сделок - в данном разделе можно добавлять  удалять этапы сделок. Устанавливать очередность.  Задавать нужный цвет каждому этапу из палитры цветов. Порядок в Виде сделки сверху вниз соответствует порядку в Сделки слева направо. Для добавления нового этапа сделки введите название этого этапа в поле Наименование.
