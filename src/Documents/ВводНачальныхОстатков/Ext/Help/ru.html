Документ "Ввод начальных остатков"

Документ предназначен для первоначального ввода остатков. Переход к списку документов производится из раздела "Настройки" / Группа "Прочее" / "Ввод начальных остатков".

В документе необходимо заполнить значения следующих полей:

  • Клиника. В этом поле необходимо указать клинику, ввод остатков для которой осуществляется. 

Документ имеет несколько табличных частей, которые располагаются на соответствующих закладках.
Информация о сертификатах в обращении, отображается на закладке "Сертификаты в обращении".

  • Сертификаты в обращении. Табличная часть:
    • Сертификат. В данное поле вводится элемент справочника номенклатуры.
    • Серийный номер. Указывается серийный номер сертификата.
    • Пациент. Указывается клиент - владелец сертификата. 
    • Количество. Количество сертификатов.
    • Сумма. Сумма остатка по сертификатам.

Информация об остатках денежных средств на лицевых счетах, отображается на закладке "Денежные средства на лицевых счетах".

  • Денежные средства на лицевых счетах. Табличная часть:
    • Пациент. Указывается клиент, владелец лицевого счета. 
    • Вид лицевого счета. Выбирается из справочника виды лицевых счетов. 
    • Сумма. Сумма остатка на лицевом счете.

Информация по продажам, необходимая для расчетов накопительных скидок, отображается на закладке "Продажи".

  • Продажи. Табличная часть:
    • Номенклатура. В данное поле вводится элемент справочника номенклатуры.
    • Характеристика, серия. Указывается характеристика номенклатуры.
    • Сотрудник. Указывается сотрудник, ответственный за продажу.
    • Ассистент. Указывается сотрудник, ассистирующий при приеме.
    • Пациент. Пациент, на кого осуществляли продажу.
    • Плательщик. Клиент - плательщик.
    • Номер зуба. Указывается номер зуба пациента.
    • Количество. Количество продаж.
    • Сумма. Сумма продаж.
    • Стоимость без скидки. Сумма продаж без скидки.

Информация по взаиморасчетом с поставщиками, отображается на закладке "Взаиморасчеты с поставщиками".

  • Взаиморасчеты с поставщиками. Табличная часть:
    • Контрагент. Поставщик, перед кем образовалась задолженность.
    • Документ реализации. Указывается документ реализации.
    • Документ аванса. Указывается документ аванса.
    • Сумма. Сумма взаиморасчетов.

Информация по расчетам с персоналом, отображается на закладке "Расчеты с персоналом".

  • Расчеты с персоналом. Табличная часть:
    • Сотрудник. Сотрудник, перед кем образовалась задолженность.
    • Месяц начисления. Указывается месяц начисления заработной платы.
    • Документ начисления. Указывается документ начисления заработной платы.
    • Сумма. Сумма начисления.
