Документ "Перемещение запасов"

Документ предназначен для оформления передачи запасов со склада на склад, со склада на сотрудника, либо с сотрудника на склад. Переход к списку документов производится из раздела "Запасы" / Раздел "Перемещение, передача" / "Перемещения запасов". 

При создании нового документа необходимо заполнить следующие реквизиты:

  • Вид операции. Необходимо выбрать вид операции из перечисленных: перемещение между складами, передача персоналу, возврат от персонала, перемещения между персоналом.
  • Структурная ед. Структурная единица отправитель. 
  • Структурная ед. получатель. Структурная единица получатель. 
  • Источник перемещения. Группа реквизитов:
    • Склад отправитель. В этом поле необходимо указать склад, из которой будут списаны запасы. Становится доступным для указания при выбранном виде операции - перемещение между складами или передача персоналу. Выбирается из справочника "Склады".
    • Персонал отправитель. В этом поле необходимо указать сотрудника с которого будут списаны запасы. Становится доступным для указания при выбранном виде операции - возврат от персонала или перемещения между персоналом. Выбирается из справочника "Сотрудники".
  • Приемник перемещения. Группа реквизитов:
    • Склад получатель. В этом поле необходимо указать склад получатель, в который будут оприходованы запасы. Становится доступным для указания при выбранном виде операции - перемещение между складами или возврат от персонала. Выбирается из справочника "Склады".
    • Персонал получатель. В этом поле необходимо указать сотрудника на которого будут перемещены запасы. Становится доступным для указания при выбранном виде операции - передача персоналу или перемещения между персоналом. Выбирается из справочника "Сотрудники".
  • Запасы. Табличная часть:
    • Номенклатура. В данное поле вводится элемент справочника номенклатуры.
    • Характеристика, серия. Указывается характеристика номенклатурной позиции. Поле доступно только в том случае, если при настройке параметров учета указано ведение учета по партиям номенклатуры. Выбирается из справочника "Характеристики, серии".
    • Количество. Указывается количество единиц измерения номенклатурной позиции.
    • Упак. Упаковка номенклатуры.  

 

  • Комментарий. Произвольный комментарий к документу.
