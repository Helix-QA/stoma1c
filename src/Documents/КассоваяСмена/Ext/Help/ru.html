Документ "Кассовая смена"

Документ предназначен для просмотра информации по кассовым сменам различных касс.

Список кассовых смен может быть отобран по кассовым сменам конкретной кассы, либо же по статусу кассовой смены. Для этого в соответствующих полях необходимо задать значения отбора.

При помощи данного документа нельзя открыть или закрыть кассовую смену.
