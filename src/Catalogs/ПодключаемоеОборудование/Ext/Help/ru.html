Справочник "Внешнее оборудование"

Данный справочник предназначен для задания и редактирования свойств внешнего торгового оборудования.

Для каждого подключаемого торгового оборудования необходимо указать значения следующих полей:

  • Наименование - произвольное строковое наименование. Программа автоматически формирует наименование на основании указанного для торгового оборудования драйвера, однако при желании его можно изменить.
  • Тип оборудования - выбирается один из следующих типов внешнего торгового оборудования:
    • Сканеры штрихкода;
    • Фискальные регистраторы;
    • Считыватели магнитных карт;
    • Эквайринговые терминалы;
    • Терминалы сбора данных;
    • IP-Телефония.
  • Обработчик драйвера - из списка доступных обработок выбирается та, которая будет выполнять функции драйвера данного торгового оборудования.
  • Модель устройства - при необходимости указывается также модель торгового оборудования.
  • Рабочее место - из списка рабочих мест выбирается то рабочее место, к которому будет подключено данное внешнее оборудование.
