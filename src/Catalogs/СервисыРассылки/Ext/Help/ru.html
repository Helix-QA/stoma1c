Справочник "Сервисы рассылки"

Данный справочник предназначен для управления рассылкой SMS-сообщений при помощи различных сервисов автоматизированной рассылки SMS-сообщений.

Каждый элемент данного справочника предназначен для какой-либо одной учетной записи на сервисах автоматизированной отправки SMS. Если у пользователя имеется зарегистрированный аккаунт на каком-либо сервисе, то для этого аккаунта необходимо создать свой элемент справочника «Сервисы SMS-рассылки», ввести логин и пароль для данного сервиса и указать обработку

Если же пользователю потребуется отправлять SMS-сообщения с другой учетной записи на данном сервисе, либо же при помощи какого-то другого сервиса отправки, в этом случае потребуется создать другой элемент справочника.

Взаимодействие программы с сервисами рассылки SMS-сообщений происходит при помощи специальных обработок, которые отличаются для каждого сервиса. По умолчанию в программе уже включены обработки для взаимодействия с большинством самых популярных сервисов SMS-рассылок. Поэтому в большинстве случаев пользователю необходимо просто выбрать из списка нужный сервис и указать в полях «Логин» и «Пароль» учетные данные для работы с выбранным сервисом. Если учетные данные для работы с выбранным сервисом отсутствуют, то можно перейти по гиперссылке, указанной в поле «Ссылка на сайт» и, зарегистрировавшись, получить нужные логин и пароль.
