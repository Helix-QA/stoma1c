Документ "Инвентаризация запасов"

Документ предназначен для отражения результатов проведения инвентаризации в производстве или на складе. Переход к списку документов производится из раздела "Запасы" / Группа "Инвентаризация" / "Инвентаризация запасов".

Документ не формирует движений. На основании него могут быть сформированы документы "Оприходование запасов" при обнаружении излишков и "Списание запасов" для списания недостач.

В документе необходимо заполнить следующие поля:

  • Вид операции. В этом поле необходимо указать вид инвентаризации - склад или персонал.
  • Вид цен. В данном поле указывается вид цен инвентаризации.
  • Склад. В этом поле необходимо указать склад в котором проводится инвентаризация.

  • Персонал. В этом поле необходимо указать сотрудника, на подотчете которого проводится инвентаризация.

  • Запасы. Табличная часть:

    • Номенклатура. В данное поле вводится элемент справочника номенклатуры.
    • Характеристика, серия. В данное поле вводится элемент справочника характеристики, серии.
    • Количество (факт). Указывается количество единиц измерения номенклатурной позиции по результатам инвентаризации.
    • Количество (учет). Указывается количество единиц измерения номенклатурной позиции по данным учета.
    • Отклонение.  Фиксируется разница между фактическим и учетным количеством запасов.
    • Упак. Упаковка номенклатуры.
    • Цена. Цена остатка запаса.
    • Сумма. Сумма остатка запаса по данным инвентаризации.
    • Сумма (учет). Сумма остатка запаса по данным учета.

 

В поле Условия проведения инвентаризации можно указать условия отбора товаров и материалов в табличной части документа.

Например, можно указать, чтобы в инвентаризации не участвовали запасы из определенных партий, либо же четко перечислить по каким именно запасам должна проводиться инвентаризация.

Комментарий. Произвольный комментарий к документу.

На основании этого документа могут быть созданы следующие документы:

  • "Оприходование запасов";
  • "Списание запасов".
