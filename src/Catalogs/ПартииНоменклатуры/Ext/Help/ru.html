Справочник "Партии номенклатуры"

Справочник предназначен для хранения информации о партиях запасов и организации партионного учета запасов.

Настройка параметров ведения партионного учета может быть осуществлена посредством формы настройки параметров учета.

Для номенклатурных позиций, принятых от сторонних контрагентов, учет по партиям обязателен.
