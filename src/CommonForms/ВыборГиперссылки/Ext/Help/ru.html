Общая форма "Выбор гиперссылки"

Данная форма предназначена для удобства указания гиперссылки.

В общем случае, для указания гиперссылки требуется указать протокол и адрес гиперссылки. Протокол выбирается при помощи левого поля.

Предусмотрены следующие варианты:

  • http://  - применяется для указания гиперссылок на любые веб-сайты. Этот вариант выбирается по умолчанию, поэтому, в большинстве случаев, достаточно просто вставить в правое поле адрес ссылки;
  • mailto:  - данный протокол применяется в тех случаях, когда при нажатии мышью по такой ссылке должна сразу же открыться форма отправки электронного письма на указанный адрес. В правом поле необходимо указать адрес электронной почты.
