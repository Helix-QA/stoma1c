Отчет по выручке

Данный отчет предоставляет данные обо всех движениях денежных средств выбранной кассы за указанный в отчете период.

Сводный отчет по выручке

Данный мини-отчет предоставляет информацию о том, на какую сумму врачами были оказаны услуги, и какая часть этой суммы погашена пациентами. Кроме этого приводится детализация по видам оплат, при помощи которых пациентами производилась оплата.

Прочие поступления и списания

В данном мини-отчете собраны данные по всем поступлениям и списаниям денежных средств, не связанных с лечением пациентов.

Наличные в кассе

Данный мини-отчет по своей сути является ведомостью наличных денежных средств для выбранной кассы.
