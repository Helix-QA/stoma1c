Отчет "Рейтинг сотрудников"

Отчет "Рейтинг сотрудников" предоставляет сведения о показателях продаж всех сотрудников клиники, а также позволяет определить сотрудников, имеющих самые лучшие показатели продаж

Настройка отчета производится посредством выбора одного из вариантов отчета, либо при нажатии кнопки "Настройка".

Для данного отчета доступны следующие варианты сортировки рейтинга:

  • По количеству обслуженных пациентов. В данном варианте в самом верху списка будут сотрудники, имеющие за указанный период наибольшее количество обслуженных пациентов.
  • По количеству приёмов. В данном варианте в самом верху списка будут сотрудники, имеющие за указанный период наибольшее количество проведенных приёмов.
  • По сумме приёмов. В этом случае вверху будут те сотрудники, которые имеют наибольшую сумму приёмов за указанный период.
  • По средней сумме приёма. В начале списка будут те сотрудники, имеющие наилучшие показатели средней суммы приёма. Показатель средней суммы приёма вычисляется как отношение суммы приёмов сотрудника за данный период и общего количества приёмов за тот же период.
