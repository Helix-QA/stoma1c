Справочник "Сотрудники"

Справочник «Сотрудники» предназначен для хранения информации о сотрудниках стоматологии.

В справочнике можно объединять сотрудников в произвольные группы и подгруппы, например, руководство, врачи, администраторы и т.д.

Редактирование и ввод новой информации о сотруднике производится в отдельном диалоговом окне.

При заполнении карточки сотрудника необходимо указать значения следующих полей:

  • Наименование сотрудника. Значение, указанное в этом поле, будет использоваться впоследствии для представления данного сотрудника в документах, печатных формах, отчетах и др.
  • Должность. Указывается должность сотрудника. Значение должности выбирается из справочника «Должности».
  • Клиника. В этом табличном поле необходимо перечислить все клиники в которых работает данный сотрудник.
  • Дата рождения. В этом поле может быть указана дата рождения сотрудника.
  • Вид цен. Программа позволяет назначить каждому сотруднику индивидуальный вид цен для оказываемых услуг и реализуемых товаров.
  • Использовать предварительную запись. Данный флажок отвечает за то, будет ли данный сотрудник доступным в Журнале записи на Рабочем столе программы, а также, собственно, за то, возможно ли будет оформить для данного сотрудника предварительную запись.
  • Списывать. Данное поле определяет то, откуда будут списываться материалы (т.н. материалы списания), расходуемые при оказании услуг. Вариант По сотруднику означает, что расходные материалы будут списываться из числа материалов, выдаваемых лично сотруднику. По складу - материалы будут списываться из склада организации.
