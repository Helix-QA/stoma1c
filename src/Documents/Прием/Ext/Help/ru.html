Документ "Прием"

Документ "Прием" предназначен для оформления приема пациента. Переход к списку документов производится из раздела "Оказание услуг" / Раздел "Оказание услуг" / "Приемы". 

В документе необходимо указать значения следующих полей:

  • Пациент. В этом поле необходимо указать пациента, которому будет проведено лечение. Значение выбирается из справочника "Клиенты".
  • Страховой полис. В этом поле необходимо указать страховой полис, по которому будут производиться взаиморасчеты.
  • Врач. В этом поле необходимо указать сотрудника организации, который будет осуществлять лечение.
  • Ассистент. В этом поле необходимо указать сотрудника организации, который будет осуществлять ассистирование.
  • Результат приема. В этом поле необходимо указать результат приема - санировано либо продолжает лечение.
  • Вызвать на профосмотр. Флаг необходимости вызова на профосмотр. Период через который необходимо вызвать указывается в поле Через.

 

  • Зубная формула. Табличная часть:
    • Зубная формула. В этом поле представлена графическая схема зубной формулы.
    • Текущий прием. Указывается статус текущего приема.
  • Обследование. Табличная часть:
    • Шаблон обследования. В данное поле вводится элемент справочника шаблоны обследований.
  • Лечение. Табличная часть:
    • Зуб. В данное поле вводится элемент перечисления Зубы.
    • Услуга. В данное поле вводится элемент справочника номенклатуры.
    • Кол-во. Указывается количество единиц измерения номенклатурной позиции.
    • Цена - цена реализуемой номенклатурной позиции.
    • % ручн. скидки - % ручной скидки по строке.
    • Сумма авт. скидки - сумма автоматической скидки по данной строке.
    • Сумма - стоимость номенклатуры по данной строке.
    • Материалы. По данной ссылке открывается форма выбора списываемых в результате приема материалов.

    • Документ аванса. В данном поле вводится документ аванса - План лечения.

  • Товары. Табличная часть:
    • Номенклатура. В данное поле вводится элемент справочника номенклатуры.
    • Характеристика, серия. Указывается характеристика номенклатурной позиции. Поле доступно только в том случае, если при настройке параметров учета указано ведение учета по характеристикам номенклатуры. Выбирается из справочника "Характеристики".
    • Кол-во. Указывается количество единиц измерения номенклатурной позиции.

    • Упаковка. - упаковка номенклатуры, если есть. Выбирается из справочника "Упаковки номенклатуры".

    • Цена - цена реализуемой номенклатурной позиции.
    • % ручн. скидки - % ручной скидки по строке.
    • Сумма авт. скидки - сумма автоматической скидки по данной строке.
    • Сумма - стоимость номенклатуры по данной строке.

    • Документ аванса. В данном поле вводится документ аванса - Наряд-заказ.

  • Скидки и наценки. Табличная часть. Указаны все автоматически рассчитанные скидки по данному Приему. 
  • Дополнительно. Табличная часть. Содержит служебные поля. 

 

На основании данного документа могут быть оформлены следующие документы:

  • "Возврат от пациента";
  • "Списание задолженности";
