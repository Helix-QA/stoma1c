Справочник "Внешние системы"

Справочник предназначен для хранения информации о внешних системах, подключаемых посредством API.

Для внешней системы необходимо указать следующие реквизиты:

  • Наименование - название внешней системы.
  • Ключ приложения - ключ доступа внешней системы.
  • Используется - флаг использования внешней системы.
  • Тип внешней системы - предопределенный тип внешней системы.
  • Синхронизирована - признак того, что внешняя система успешно синхронизирована.
  • Управление в личном кабинете - признак управления в личном кабинете.
  • Авторизация по коду без СМС - при установленном флаге авторизация будет проходить без подтверждения смс.
