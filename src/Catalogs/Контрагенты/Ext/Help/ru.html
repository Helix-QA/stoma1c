Справочник "Контрагенты"

Справочник предназначен для хранения списка контрагентов. Контрагент - это общее понятие, куда включены поставщики и покупатели, организации и частные лица.

 

Контрагенты могут быть сгруппированы по группам справочника, например, "Поставщики", "Лаборатории" , "Пациенты", "Страховые компании". Справочник контрагентов имеет неограниченное количество уровней вложенности.

В программе применяется разделение контрагентов на пациентов и прочих контрагентов, таких как поставщики, банки и др. В зависимости от того, является контрагент клиентом или нет определяется список реквизитов, которые необходимо указать для контрагента.

Пациенты

Для пациентов необходимо указать следующую информацию:

  • Группа клиентов. Группа справочника Контрагенты, к которой относится данный клиент.
  • Фамилия, имя, отчество. На основании ФИО пациента программа автоматически формирует представление пациента.
  • Дата рождения. Дата рождения пациента.
  • Пол. Пол пациента.
  • Удостоверение личности. Группа полей, в которых указываются удостоверение личности пациента.
  • Социальный статус. Группа полей, в которых указываются данные о социальном статусе пациента, месте работы.
  • Маркетинг. В группе маркетинг представлены данные, позволяющие указать тот способ, благодаря которому пациент узнал о стоматологии.
  • Полисы медицинского страхования. На данной закладке указываются полисы медицинского страхования пациента.
  • Скидки пациента. На данной закладке указываются скидки пациента.
  • Дополнительная информация. Указывается различная дополнительная информация о клиенте, например, его возрастная группа.

 

Прочие контрагенты

Для всех прочих контрагентов указывается следующая информация:

  • Юр. / физ. лицо. Тип контрагента. Указывается один из вариантов: юр. лицо или физ. лицо.
  • Полное наименование контрагента. Используется для отражения в печатных формах документов.
  • Дата рег. Дата регистрации данного контрагента.
  • Группа контрагентов. Группа контрагентов указывает на ту группу справочника, в которую будет помещен текущий контрагент.

 

На закладке "Общие" указываются общие реквизиты контрагента:

  • Коды. Группа реквизитов:
    • ИНН - идентификационный номер контрагента, для отражения в печатных формах документов.
    • КПП - код постановки на учет. Для отражения в печатных формах документов. Указывается только для юридического лица.
    • ОКПО - код по Общероссийскому классификатору предприятий и организаций, для отражения в печатных формах документов.
  • Значения по умолчанию. Группа реквизитов:
    • Банковский счет по умолчанию, для подстановки в документы. Выбирается из справочника "Банковские счета".

 

На закладке Адреса, телефоны указывается контактная информация контрагента: E-mail, другое, почтовый адрес, телефон, факс, фактический адрес и юридический адрес.

На закладке Дополнительная информация указываются любая дополнительная информация о контрагенте.

При нажатии кнопки Дополнительные реквизиты открывается диалоговое окно редактирования дополнительных реквизитов, определяемых пользователем.
