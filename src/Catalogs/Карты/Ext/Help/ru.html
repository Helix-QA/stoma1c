Справочник "Карты"

Данный справочник предназначен для хранения сведений об информационных картах.

Для каждой информационной карты необходимо указать значения следующих полей:

  • Владелец карты - указывается контрагент, владеющий данной картой;
  • Дата открытия - дата, начиная с которой данная карта считается открытой;
  • Вид карты - указывается, является ли карта штриховой или магнитной;
  • Код карты - в данном поле необходимо указать код карты;
  • Наименование - наименование карты можно ввести вручную, либо выбрать один из сгенерированных программой вариант наименования.
