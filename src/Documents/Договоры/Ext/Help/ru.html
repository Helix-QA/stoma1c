Документ "Договор с пациентом"

Документ предназначен для оформления договора с пациентом клиники. Переход к списку документов производится из раздела "Справочная информация" / Группа "Контрагенты" / "Договоры с пациентами".

В документе необходимо заполнить значения следующих полей:

  • Структурная единица. В этом поле необходимо указать структурную единицу организации, в которой заключается договор с клиентом.
  • Шаблон договора. В данное поле вводится элемент справочника Шаблоны договоров.
  • Дата. Поле, в котором указывается дата заключения договора.
  • Пациент. В данное поле вводится элемент справочника Контрагенты.  
  • Врач. Поле, в котором указывается врач проводящий лечение.
  • Законный представитель. Поле, в котором указывается законный представитель пациента.
  • Комментарий. В данное поле вводится комментарий к договору.
