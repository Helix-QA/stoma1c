Обработка "Импорт справочников"

Данная обработка предназначена для загрузки в программу данных по номенклатуре или клиентам. Загрузка производится из внешних файлов (которые могут содержать данные, предварительно выгруженные из других программ).

Для начала процесса импортирования данных необходимо выбрать в поле "Загружаемый справочник" в какой именно справочник будут загружаться данные, в справочник "Номенклатура" или "Контрагенты".

После этого, в поле "Папка справочника" нужно выбрать ту папку справочника, в которую будут помещены загруженные данные. Если нужной папки еще нет - ее можно создать на данном этапе.

Тот файл, из которого будут импортироваться данные, должен быть указан в поле "Файл". Импорт данных возможен из файлов Excel (*.xls, *.xlsx), а также специальных текстовых файлов с разделителями (*.csv).

После указания файла для импортирования программа производит анализ колонок, которые имеются в файле. На это может потребоваться некоторое время.

Важно! При загрузке данных из электронных таблиц Excel (файлы *.xls, *.xlsx) убедитесь, что нужные для загрузки данные располагаются на первом листе!

Также, перед началом процесса импорта данных необходимо в соответствующей таблице установить соответствие между реквизитами справочника и колонками справочника. Для этого нужно напротив реквизита справочника выбрать ту колонку файла, из которой эти данные будут браться.

После того как все необходимые параметры импорта будут заданы, можно запустить процесс загрузки данных при помощи кнопки Загрузить.

Важно! При повторной загрузке данных из того же файла программа опять загрузит все данные в справочник. Поэтому, для того чтобы в справочнике не создавались одинаковые элементы, рекомендуется производить импорт из одного файла только один раз.
