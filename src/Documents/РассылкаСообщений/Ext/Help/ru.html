Документ "Рассылка сообщений"

Документ предназначен для организации рассылки клиентам организации SMS-сообщений с напоминаниями о предстоящем посещении, информации о новых акциях и др.

Для отправки SMS-сообщений необходимо заполнить текст сообщения на вкладке «Содержание». Текст сообщения можно на основе созданного ранее шаблона SMS-сообщений, для этого необходимо нажать кнопку "Подбор по шаблону" и выбрать подходящий шаблон из справочника "Шаблоны SMS-сообщений".

При помощи кнопки "Транслитерация" введенный текст сообщения можно транслитерировать, т.е. произвести конвертацию символов кириллицы соответствующими символами латинского алфавита и наоборот.

На вкладке «Получатели» документа необходимо указать всех получателей данной рассылки. Обратите внимание, что у всех получателей необходимо правильно указать номера мобильных телефонов, в противном случае доставка им сообщений не гарантируется.

Сервисы SMS-рассылки

Отправка SMS-сообщений производится при помощи сервисов автоматизированной рассылки. Взаимодействие программы с различными сервисами SMS-рассылки происходит при помощи специальных обработок.

Важно. Для того чтобы рассылка SMS-сообщений с помощью данного документа производилась корректно, необходимо предварительно настроить параметры рассылки SMS-сообщений. Настройка параметров рассылки производится при помощи справочника "Сервисы SMS-рассылки".

 

В поле "Сервис рассылки"  необходимо выбрать среди элементов справочника "Сервисы SMS-рассылки" тот, который отвечает за взаимодействие с необходимым сервисом.

Отправка сообщений

Отправка SMS-сообщения всем указанным в документе получателям производится при помощи нажатия кнопки "Отправить SMS". При этом есть два варианта отправки SMS-сообщений:

  • Немедленная отправка. Как и следует из названия, этот вариант предполагает немедленное начало отправки сообщений указанным в документе получателям.
  • Отправка в указанное время. При выборе этого варианта, отправка сообщений начнется в указанное время. Если в назначенное время программа будет выключена, тогда отправка сообщений начнется сразу же при следующем включении программы.

Важно. Отправка SMS-сообщений в назначенное время будет возможна только в том случае, если на форме настройки параметров учета установить значение сервиса SMS-рассылки по умолчанию.
