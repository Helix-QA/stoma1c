Форма настройки параметров учета

Форма настройки параметров учета - специальная форма, предназначенная для настройки параметров, по которым будет вестись учет в стоматологии.

Данная форма вызывается из панели "Сервис" подсистемы "Администрирование".

На форме настроек присутствуют следующие закладки:

  • Общие настройки. Раздел общих настроек.
  • Запасы. Раздел настройки параметров подсистемы запасов.
    • Стратегия списания партий товаров по хронологии. Для выбора доступны следующие варианты:

      • ФИФО (FIFO) – при установке этого значения первыми списываться будут товары тех партий, которые поступили на склад раньше других.

      • По средней – при установке данного варианта стоимость запасов будет рассчитываться как средняя стоимость всех запсов данного вида.

  • Взаимоотношения с клиентами. Раздел настройки параметров взаимоотношений с клиентами.

    • Сервис SMS-рассылки по умолчанию. В данном поле из значений справочника "Сервисы SMS-рассылки" необходимо выбрать тот сервис, который будет использоваться по умолчанию.
