Документ "Поступление запасов"

Документ предназначен для хранения информации по отражению в учете поступления запасов от сторонних контрагентов. Переход к списку документов производится из раздела "Запасы" / Раздел "Закупки" / "Поступления". 

 

Необходимо заполнить следующие поля документа:

  • Контрагент. В данном поле указывается информация о поставщике. Значение выбирается из справочника "Контрагенты".
  • Склад. В данном поле указывается информация о складе, на который будет оформляться поступление запасов.

 

  • Запасы. Реквизиты табличной части:

    • Номенклатура - позиция номенклатуры, поступление которой отражается данным документом. Выбирается из справочника "Номенклатура".
    • Партия - в данном поле указывается партия номенклатуры. Выбирается из справочника "Партии номенклатуры".
    • Количество - количество поступившей номенклатурной позиции.

    • Упаковка. - упаковка номенклатуры, если есть. Выбирается из справочника "Упаковки номенклатуры".

    • Цена - цена поступившей номенклатурной позиции.

    • Сумма - стоимость поступившей номенклатуры по данной строке.

    • Ставка НДС - ставка налога на добавленную стоимость.

    • Сумма НДС - сумма налога на добавленную стоимость по данной строке.

 

  • Комментарий - поле ввода, предназначенное для указания любой дополнительной информации к документу.

 

Данный документ может быть создан на основании документа "Заказ поставщику".

На основании этого документа могут быть созданы следующие документы:

  • "Возврат поставщику";
  • "Списание денежных средств".
