Отчет "Ведомость по реализованным сертификатам"

Отчет предоставляет сведения обо всех подарочных сертификатах, которые были реализованы клиентам в течение указанного в отчете периода.

В отчете предоставляются сведения о наименовании абонемента, его серийном номере (если для абонемента предусмотрено ведение по серийным номерам), а также структурная единица, где сертификат был реализован.

По каждому из сертификатов данные выводятся в следующих группах:

  • Реализовано - сведения о количестве и номинале реализованных сертификатов;
  • Погашено - сведения о количестве и части номинала, которые были погашены;
  • К погашению - сведения о количестве и части стоимости подарочных сертификатов, по которым организация сохраняет обязательства перед клиентами.
