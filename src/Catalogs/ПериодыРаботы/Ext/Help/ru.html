Справочник "Периоды работы"

Данный справочник предназначен для указания периодов работы (рабочих смен), предусмотренных для сотрудников организации.

Программа позволяет назначать индивидуальные периоды работы разным структурным подразделениям организации.

При вводе периода работы необходимо указать время начала и окончания данного периода, а также структурную единицу организации, для которой данные период предназначен. После этого программа автоматически заполнит наименование (представление) данного периода на основании введенных данных. В случае если предложенное программой наименование по каким-либо причинам не подходит, его можно отредактировать в любой момент.
