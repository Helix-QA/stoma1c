Обработка "Выгрузка данных в конфигурацию "Бухгалтерия предприятия", ред 2.0" предназначена для выгрузки данных в файл из текущей конфигурации в конфигурацию "Бухгалтерия предприятия", ред 2.0".

Выгрузка данных

Для осуществления выгрузки данных необходимо указать имя файла, в который будет осуществляться выгрузка данных.

Для выгрузки документов и записей независимых периодических регистров сведений необходимо указать период - "Дату начала" и "Дату окончания". Результирующий файл с выгруженными данными может быть сжат и установлен пароль.

На закладке "Выгружаемые данные" можно выбрать те типы объектов, которые должны выгружаться.

На закладке "Параметры выгрузки" можно указать дополнительные параметры выгрузки данных.

На закладке "Комментарий" можно написать произвольный текст-комментарий, включаемый в файл обмена.
