Отчет "Отчет по клиентам"

Отчет предоставляет разнообразные данные по клиентам организации.

Настройка отчета производится посредством выбора одного вариантов отчета, либо при нажатии кнопки "Настройка".

Варианты отчета

Выбор варианта отчета возможен путем нажатия кнопки "Выбор варианта". Текущий вариант отчета отображается в поле "Вариант отчета".

Для данного отчета доступны следующие варианты:

  • Отчет по клиентам. Предоставляются персональные данные по всем клиентам, которые были зарегистрированы в указанный период.
  • Дни рождения. Предоставляются данные о днях рождения клиентов.
  • Новые клиенты по сотрудникам. В разрезе пользователей информационной базы предоставляется информация о клиентах зарегистрировавшихся в указанный период.
  • Новые клиенты по дням. Данные представляются в виде гистограммы, отображающей количество зарегистрированных клиентов за каждый отдельно взятый день.
  • Новые клиенты по неделям. Данные представляются в виде гистограммы, отображающей количество зарегистрированных клиентов за каждую неделю.
  • Новые клиенты по месяцам. Данные представляются в виде гистограммы, отображающей количество зарегистрированных клиентов за каждый месяц.
