Справочник "Сделки"

Справочник предназначен для хранения списка сделок по пациентам. Отражает поэтапный процесс стадии сделки. В итоге должен быть конечный результат, где может быть успех, либо отказ. При успехе, выбирается реализация, а при отказе - причина отказа.

 

В данный справочник вносится следующая информация:

 

Этапы сделки. Отражает текущее состояние сделки. Серым цветом указываются неактивные этапы. Цветные - выполненные. Если после цветного этапа, установлен серый этап, значит последний цветной, является текущим этапом сделки. При выборе успеха(реализации), все этапы и сделка в целом, считаются завершенными. Если выбран отказ, тогда предыдущие этапы обнуляются, и сделка считается так же завершенной.
  • Наименование - имя сделки.
  • Сумма сделки - сумма, на которую заключилась сделка.
  • Канал привлечения - канал привлечения пациента.
  • Рекламный источник - в данном поле необходимо указать откуда пациент узнал о клинике. Можно указать в качестве источника другого пациента клиники. В этом случае необходимо будет указать этого пациента, выбрав его из справочника пациентов.
  • Заявка - в автоматическом порядке указывается заявка из которой, была создана сделка.
  • Менеджер - выбирается из справочника сотрудников. Заполняется автоматически текущим пользователем программы при создании заявки.
  • Описание - краткое описание сделки.
  • Фамилия, имя, отчество (ФИО)- указывается фамилия, имя и отчество потенциального пациента. На основании ФИО программа автоматически формирует представление потенциального пациента.
  • Контакты клиента. Группа контактных данных потенциального пациента:
    • Телефон - основной телефон потенциального пациента.
    • Email - электронный адрес потенциального пациента. По нажатию на кнопку  можно отправить письмо непосредственно из программы.
    • Адрес - адрес проживания потенциального пациента. Используется при печати договора.
    • Рассылка разрешена (рассылка запрещена) - по нажатию на ссылку происходит переключение флага разрешения отправки потенциальному пациенту смс-рассылок.
