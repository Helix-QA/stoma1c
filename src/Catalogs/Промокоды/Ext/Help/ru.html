Справочник "Промокоды"

Справочник предназначен для хранения информации, состоящей из букв и/или цифр совокупности символов, дающей возможность установить скидку с помощью маркетинговой акции и начислить бонусные баллы пациенту, с помощью которого было привлечение нового пациента в клинику.

Для промокода необходимо заполнить следующие реквизиты:

  • Промокод - краткое наименование промокода.
  • Реферальная программа - тумблер устанавливающий признак реферальной программы. В выключенном положении указывает на то что промокод применяется в качестве скидки по маркетинговой акции без начисления бонусных баллов рефералам.
  • Рефералы - выбор варианта реферальной программы. Возможные значения - пациенты либо сотрудники.
  • Сегмент рефералов - выбор сегмента сотрудников либо пациентов, которые могут использовать данный промокод.
  • Длина промокода - указывается длина реферального промокода в количестве символов.
