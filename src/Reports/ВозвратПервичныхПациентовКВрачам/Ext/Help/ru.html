Отчет "Возврат первичных пациентов к врачам"

Отчет анализирует и выдает данные по возврату первичных пациентов к врачу.

Колонки:

  • Первичные для клиники пациенты - количество первичных пациентов (один пациент может быть только один раз первичным для Клиники)
  • Первичные для врача пациенты
    • Пациентов - количество первичных пациентов именно этого врача. (один пациент может быть первичным для нескольких врачей одновременно)
    • Не вернулись - пациенты пришедшие на прием только один раз. (В будущем они еще могут вернуться, но на данный момент времени они считаются не вернувшимися)
    • Вернулись - пациенты пришедшие на повторные приемы именно к этому врачу!
      • Среди не вернувшихся, вернулись - информация о количестве повторных приемов для каждого  из врачей.
