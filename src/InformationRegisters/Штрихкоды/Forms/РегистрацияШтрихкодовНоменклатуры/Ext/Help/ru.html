Регистрация штрихкодов номенклатуры

 

В каких случаях открывается форма "Обнаружены неизвестные штрих коды"?

Форма "Обнаружены неизвестные штрих коды" открывается если считываемый штрих код не зарегистрирован в информационной базе. В форме программа предложит зарегистрировать новый штрих код и сопоставить его имеющемуся в информационной базе товару.

Как зарегистрировать новый штрих код?

Для того что бы зарегистрировать новый штрих код в информационной базе его необходимо сопоставить с имеющимся в базе товаром. Для этого в поле "Номенклатура" необходимо выбрать товар который соответствует новому штрих коду, заполнить характеристику товара (если используется), указать для какой единицы измерения регистрируется новый штрих код. По команде "Зарегистрировать штрих коды" новый штрих коды будут зарегистрированы, форма закроется. Закрыть форму без регистрации новых штрих кодов можно с помощью команды "Не регистрировать"
