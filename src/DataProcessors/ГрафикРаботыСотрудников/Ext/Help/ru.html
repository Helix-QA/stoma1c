Обработка "График работы сотрудников"

Данная обработка предназначена для планирования графика работы сотрудников.

Заполнение графика работы сотрудников

При заполнении графика работы сотрудников, прежде всего, необходимо указать структурную единицу организации в поле «Структурная единица». Это определит доступные периоды работы в соответствии с указанной структурной единицей.

Также необходимо сразу указать в поле «Период» тот период, за который производится редактирование.

Дальнейшее заполнение графика работы сотрудников лучше всего производить следующим образом:

  1. Из списка доступных периодов работы выбрать в поле «Выбранные периоды работы» тот период (или те периоды) которые необходимо назначить сотруднику или группе сотрудников.

     

    Примечание. Несколько периодов могут быть выбраны, например, в том случае, когда необходимо определить полный рабочий день с учетом нерабочего обеденного времени. В этом случае будет достаточно выбрать два периода работы: до обеда и после обеда. Например, периоды «9:00 – 12:00» и «13:00 – 18:00» позволят назначить рабочий день с 9:00 до 18:00 с перерывом на обед с 12:00 до 13:00.
     
  2. В поле «Сотрудники» выбрать тех сотрудников, для которых будет редактироваться график работы.

     

    Примечание. Можно выбрать всех сотрудников одной должности просто щелкнув мышкой по нужной должности.
     
  3. В табличном поле «Отмеченные дни» двойным щелчком мыши в ячейке дня можно определить сотруднику рабочий день, состоящий из выбранных в поле «Выбранные периоды работы» периодов работы.

     

    Примечание. Если какой-либо день в графике работы сотрудника помечен как рабочий, сделать его нерабочим можно следующим образом: Очистить поле «Выбранные периоды работы», т.е. сделать так, чтобы не был выбран ни один период. Затем дважды щелкнуть мышкой на тот день (или те дни), которые необходимо отметить как нерабочие дни.
