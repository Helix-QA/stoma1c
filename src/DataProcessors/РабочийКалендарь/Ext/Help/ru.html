Рабочий стол

Рабочий стол – это основной рабочий инструмент администратора регистратуры стоматологической клиники. Именно с рабочего стола начинается каждый сеанс работы с программой.

Основное место на рабочем столе администратора отведено Журналу записи. Журнал записи предоставляет визуальное представление всех запланированных, осуществляемых и прошедших событий стоматологии, как правило, посещений пациентов. Позволяет просмотреть прошедшие события, запланировать новые, а также изменить или отменить запланированные события.

Журнал записи может быть представлен в нескольких вариантах. Переключение вариантов отображения производится при помощи соответствующих кнопок, расположенных над самим журналом.

В программе предусмотрено два варианта оформления, рассмотрим каждый из них.

День.     В этом варианте область представлена колонками и строками, на пересечении которых можно оформить событие. В колонках указываются сотрудники, к которым разрешена предварительная запись и для которых данный день указан как рабочий. В строках указываются часы (одна строка определяет пятнадцатиминутный интервал времени).

Неделя. Данный режим предназначен для наглядного отображения событий стоматологии за неделю. Настройка состава недели производится на форме настроек параметров журнала, открыть эту форму можно при помощи кнопки «Настройка» в верхнем правом углу экрана.

Помимо перечисленных вариантов представления журнала записи, события в нем могут быть легко отсортированы по любому сотруднику, либо по любой должности сотрудников. Для этого достаточно в панели отбора событий выделить строку, с каким либо сотрудником или должностью сотрудников.

Настройка цветов событий

Настройка цветов событий производится при помощи группы кнопок в верхней части журнала записи. Можно отдельно настроить представление оплаченных, неоплаченных и просроченных событий. Щелчком левой клавиши мыши по цветному прямоугольнику напротив нужного вида событий можно открыть форму настройки цветов.

Оформление предварителной записи

Оформление предварительной записи производится нажатием правой клавиши мыши в области журнала записи. На экране появится форма документа предварителной записи. При этом время начала предварителной записи будет соответствовать времени выбранной строки журнала записи.
