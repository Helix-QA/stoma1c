Отчет "Оказанные услуги, продажи"

Данный отчет предоставляет данные по продажам организации.

Настройка отчета производится посредством выбора одного вариантов отчета, либо при нажатии кнопки "Настройка".

Варианты отчета

Выбор варианта отчета возможен путем нажатия кнопки "Выбор варианта". Текущий вариант отчета отображается в поле "Вариант отчета".

Для данного отчета доступны следующие варианты:

  • Оказанные услуги, продажи. Данные предоставляются в разрезе номенклатуры.
  • Оказанные услуги, продажи по персоналу. Предоставляются данные в разрезе  сотрудников и проданной номенклатуры.
  • Оказанные услуги, продажи по пациентам. Данные предоставляются в разрезе покупателей и проданной номенклатуры.
