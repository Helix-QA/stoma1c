Справочник "Виды бонусных счетов"

Справочник служит для указания всех возможных видов бонусных счетов клиентов.

Для вида бонусного счета необходимо заполнить следующие реквизиты:

  • Наименование - краткое наименование бонусного счета.
  • Именной | на предъявителя - переключатель ограничения по пользователю бонусного счета.
  • Начисления. Группа реквизитов:
    • Приветственное - сумма приветственного начисления, при открытии бонусного счета.
    • В день рождения - сумма начисления бонусов в день рождения клиента. 
    • Не начислять при оплате бонусами – при оплате бонусом с данного вида бонусного счета маркетинговые акции по начислению бонусов применяться не будут.
    • Списание с бонусного счета - выбор ограничения списания с помощью бонусного счета. Может принимать два значения - не ограничено либо ограничено процентом списания. С помощью данного параметра можно установить, например, возможность погашения бонусами только половины стоимости покупки.
    • Срок действия бонусного счета - выбор периода действия бонусного счета. Может принимать три значения - бессрочный, ограничен периодом в интервале дат, действует заданный срок в днях, месяцах либо годах. В случае установки параметра срока действия бонусный счет будет закрываться по прошествии срока его действия.
    • Срок действия начислений - выбор периода действия бонусов на счете. Может принимать два значения - бессрочный либо ограничен периодом. В случае установки параметра срока действия бонусы будут сгорать по прошествии срока их действия.
  • Доступная номенклатура - таблица с выбором номенклатуры либо сегментов, возможных для погашения данным видом бонусного счета.
