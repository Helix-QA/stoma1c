Справочник "Состояния зубной формулы"

Справочник предназначен для хранения всех состояний зубов и десен, которые могут устанавливаться в графической зубной формуле.

Для создания нового или редактирования уже существующего состояния, прежде всего необходимо выбрать область, к которому оно будет относиться. Сделать это можно двумя способами: выбрать нужную область из разворачивающего списка Области, либо щелкнуть левой клавишей мыши по нужной области десны или зуба в области графического представления.

После выбора нужной области на экране будут представлены все уже существующие состояния для данной области.

Выбрав существующее состояние или создав новое, можно открыть форму редактирования состояния зубной формулы. Здесь необходимо указать его наименование, цвет, который будет соответствовать данному состоянию на графической зубной формуле, а также условное обозначение, т.е. тот символ, который будет соответствовать данному состоянию в табличном представлении зубной формулы.
