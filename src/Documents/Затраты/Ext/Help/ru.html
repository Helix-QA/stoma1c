Документ "Затраты"

Документ предназначен для фиксации затрат. Переход к списку документов производится из раздела "Финансы" / Группа "Затраты" / "Затраты".

Затраты должны быть указаны в таблице Затраты. При этом указываются следующие значения:

  • Период - то время когда произошла затрата;
  • Статья затрат - статья затрат, которой соответствует данная затрата;
  • Сотрудник - тот сотрудник, на которого приходится данная затрата;
  • Контрагент - указывается контрагент, с которым связана данная затрата;
  • Сумма - сумма затраты.
