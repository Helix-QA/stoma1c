Документ "Возврат от пациента"

Документ "Возврат от пациента" используется для оформления возврата товаров, проданных конкретному пациенту. Переход к списку документов производится из раздела "Оказание услуг" / Группа "Продажи товаров" / "Возвраты от пациента".

Оформление документа "Возврат от пациента" осуществляется в том случае, если пациент возвращает товар после закрытия той кассовой смены, в течение которой была оформлена реализация возвращаемых товаров.

В документе "Возврат от пациента" в поле Пациент указывается тот контрагент (покупатель), который возвращает проданный ему ранее товар. В поле Склад необходимо указать тот склад в который будут возвращены товары после проведения документа. В поле Сотрудник необходимо указать того сотрудника, который осуществлял продажу. В поле Ассистент необходимо указать того ассистента, который ассистировал в приеме. Страховой полис - полис, по которому проходили взаиморасчеты. Основание - документ приема

В табличную часть документа вносится информация о возвращаемых товарах. В качестве основания возврата указывается тот документ "Реализация", которым оформлялась продажа товаров покупателю.

Документ имеет несколько табличных частей, которые располагаются на соответствующих закладках.
Информация о возврате некачественно оказанных услуг, отображается на закладке "Лечение".

  • Лечение. Табличная часть:
    • Номенклатура. В данное поле вводится элемент справочника "Номенклатура".
    • Зуб. Указывается зуб.
    • Количество. Указывается количество упаковок номенклатурной позиции.

    • Цена - цена возвращаемой номенклатурной позиции.

    • % ручн. скидки - % ручной скидки по строке.
    • Сумма авт. скидки - сумма автоматической скидки по данной строке.

    • Сумма - стоимость номенклатуры по данной строке.

    • Документ аванса. В данном поле необходимо выбрать документ аванса.

Информация о возврате товаров, отображается на закладке "Запасы".

  • Запасы. Табличная часть:
    • Номенклатура. В данное поле вводится элемент справочника "Номенклатура".
    • Характеристика, серия. Указывается характеристика номенклатурной позиции. Поле доступно только в том случае, если при настройке параметров учета указано ведение учета по характеристикам номенклатуры. Выбирается из справочника "Характеристики, серии".
    • Количество. Указывается количество упаковок номенклатурной позиции.

    • Упак. - упаковка номенклатуры, если есть. Выбирается из справочника "Упаковки номенклатуры".

    • Цена - цена возвращаемой номенклатурной позиции.

    • % ручн. скидки - % ручной скидки по строке.
    • Сумма авт. скидки - сумма автоматической скидки по данной строке.
    • Сумма - стоимость номенклатуры по данной строке.
    • Ставка НДС - ставка налога на добавленную стоимость.

    • Сумма НДС - сумма налога на добавленную стоимость по данной строке.

    • Всего. Общая стоимость номенклатурной позиции с учетом НДС.
    • Документ аванса. В данном поле необходимо выбрать документ аванса.

 

Возврат наличных денежных средств оформляется документом "Списание денежных средств" с установленным видом операции Возврат оплаты клиенту.

После проведения документа товар возвращается на склад, указанный в документе, а стоимость возвращаемых товаров или услуг выдается покупателю наличными денежными средствами по документу "Списание денежных средств".
