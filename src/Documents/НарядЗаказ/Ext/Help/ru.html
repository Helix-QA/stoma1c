Документ "Наряд-заказ"

В документе "Наряд-заказ" фиксируется согласованный заказ изделий в стороннюю лабораторию. Переход к списку документов производится из раздела "Лечение" / Группа "Лаборатория" / "Наряд-заказы".

В документе необходимо заполнить следующие поля:

  • Лаборатория. В этом поле необходимо указать лабораторию. Выбирается из справочника "Контрагенты".
  • Склад. В поле необходимо указать тот склад, на который должны будут поступить заказываемые товары или материалы.
  • Пациент. Здесь необходимо указать пациента. Выбирается из справочника "Контрагенты".
  • Диск. карта. В этом поле указывается дисконтная карта пациента.
  • Врач. В этом поле указывается врач. Выбирается из справочника "Сотрудники".
  • Статус. В данном поле указывается статус наряд-заказа. Может принимать два значения: открыт либо закрыт.

Документ имеет табличную часть, которая располагается на закладке Запасы и услуги.

  • Запасы и услуги. Табличная часть:
    • № заказа. Указывается номер заказа поставщика.
    • Зуб. Указывается зуб.
    • Номенклатура. В данное поле вводится элемент справочника "Номенклатура".
    • Характеристика, серия. Указывается характеристика номенклатурной позиции. Поле доступно только в том случае, если при настройке параметров учета указано ведение учета по характеристикам номенклатуры. Выбирается из справочника "Характеристики, серии".
    • Количество. Указывается количество упаковок номенклатурной позиции.

    • Упаковка - упаковка номенклатуры, если есть. Выбирается из справочника "Упаковки номенклатуры".

    • Цена закупки. Цена закупки изделия.

    • Цена продажи. Цена продажи изделия пациенту.

    • % ручн. скидки - % ручной скидки по строке.
    • Сумма авт. скидки - сумма автоматической скидки по данной строке.

    • Сумма закупки. Сумма закупки изделия.

    • Сумма продажи. Сумма продажи изделия пациенту.

    • Дата заказа. Дата размещения заказа в лаборатории.
    • Дата выполнения. Планируемая дата выполнения заказа.
    • Лаборатория. В этом поле необходимо указать лабораторию. Выбирается из справочника "Контрагенты".
    • Вид цен закупки. Закупочный вид цен для данной лаборатории.
    • Статус. В данном поле указывается статус наряд-заказа. Может принимать два значения: открыт либо закрыт.
    • План лечения. Ссылка на документ план лечения.
    • Прием. Ссылка на документ прием.
  • Комментарий - поле ввода, предназначенное для указания любой дополнительной информации к документу.
