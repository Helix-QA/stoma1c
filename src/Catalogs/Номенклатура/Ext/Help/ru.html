Справочник "Номенклатура"

Справочник предназначен для хранения информации о товарах, услугах.

Справочник имеет многоуровневую иерархическую структуру. Элементы справочника можно перегруппировать, перенося их из группы в группу. При описании объекта номенклатуры в справочной системе помимо названия необходимо определить ряд дополнительных свойств. Часть из них будут в дальнейшем использоваться в качестве значений по умолчанию при подстановках в реквизиты документов, т.е. такие свойства задаются не жестко и могут быть переопределены в любой момент.

 

В данный справочник заносится следующая информация:

  • Группа. Ссылка на группу справочника, в которую включен элемент. При ручном изменении реквизита "Группа" происходит перемещение текущего элемента из старой группы в новую.
  • Краткое наименование. Используется для выбора и быстрого поиска номенклатуры из списка.
  • Код. Код элемента номенклатуры.
  • Тип. Тип номенклатуры определяет состав реквизитов элемента номенклатуры и порядок использования. Доступные типы номенклатуры:
    • Товар. Материальные запасы, подлежащие складскому учету и продаже.
    • Услуга. Услуги (сервис), например, услуги парикмахера, услуги массажиста.
    • Подарочный сертификат. Обычно это картонная или пластиковая карта, которая несет в себе некоторый эквивалент стоимости.
    • Материал. Материальные запасы, подлежащие списанию при оказании услуг.
    • Имущество. Предназначен для хранения перечня основных средств и нематериальных активов организации.
    • Набор услуг. Набор оказываемых услуг.
  • Полное наименование. Используется для вывода в печатные формы документов. Например: «EMULSIONDOR EUROTYPE Мягкий универсальный оксикрем» – это полное наименование, «EMULSIONDOR EUROTYPE» – наименование для быстрого поиска.

В зависимости от типа номенклатуры может потребоваться указать дополнительные параметры номенклатуры. Например, для услуг можно задать Нормы списания материалов, т.е. указать то количество расходных материалов, которое затрачиваются при оказании данной услуги. Кроме того, для услуг может быть задана Норма времени. Норма времени - это то количество времени (в минутах), которое обычно необходимо для оказания данной услуги.

Для подарочных сертификатов необходимо обязательно указать номинал сертификата и тип срока действия. При необходимости вести учет серийных номеров сертификатов, необходимо установить флажок Вести серийные номера.
