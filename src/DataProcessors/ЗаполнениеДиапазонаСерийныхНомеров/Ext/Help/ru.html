Обработка "Заполнение диапазона серийных номеров"

Данная обработка предназначена для формирования серийных номеров подарочных сертификатов на основании введенных параметров.

Для формирования серийных номеров необходимо указать значения следующих параметров:

  • Количество сертификатов. Указывается необходимое количество серийных номеров.
  • Префикс. Указывается префикс формируемых серийных номеров (Строка).
  • Постфикс. Указывается постфикс формируемых серийных номеров (Строка).
  • Начало диапазона. Выбирается указывается номер первого серийного номера из формируемого диапазона (Число).
  • Количество символов. Указывается количество символов для формируемых серийных номеров.

Например, если в качестве начала диапазона указать число 15, а количество символов задать равным 6-ти, тогда первый серийный номер из диапазона будет иметь числовой номер 000015.

По мере того, как будут указываться параметры для создания диапазона серийных номеров, в полях Первый код и Последний код будут отображаться, соответственно, первый и последний серийные номера из формируемого диапазона.

При нажатии кнопки Сформировать, данные серийные номера будут сгенерированы программой.
