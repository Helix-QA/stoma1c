Документ "Списание запасов"

Документ предназначен для оформления факта списания материальных ценностей со склада либо с подотчета на сотруднике. Переход к списку документов производится из раздела "Запасы" / Раздел "Инвентаризация" / "Списания запасов". 

Документ может быть оформлен на основании проведенной инвентаризации или как свободный документ в случае произвольного списания запасов.

В документе необходимо заполнить значения следующих полей:

  • Вид операции. В зависимости от вида операции документ может списывать запасы со склада или сотрудника.
  • Основание. Выбор или просмотр документа инвентаризации, на основании которого производится списание.
  • Склад. В этом поле необходимо указать склад, из которого будут списаны запасы. 
  • Статья расходов. В этом поле указывается статья расходов при списании запасов.
  • Сотрудник расхода. В этом поле указывается сотрудник при списании запасов.
  • Запасы. Табличная часть:
    • Номенклатура. В данное поле вводится элемент справочника номенклатуры.
    • Характеристика, серия. Указывается характеристика номенклатурной позиции. Поле доступно только в том случае, если при настройке параметров учета указано ведение учета по партиям номенклатуры. Выбирается из справочника "Характеристики, серии".
    • Количество. Указывается количество единиц измерения номенклатурной позиции.
    • Упак. - упаковка номенклатурной позиции.
    • Склад. В этом поле необходимо указать склад, из которого будут списаны запасы. 
    • Статья расходов. В этом поле указывается статья расходов при списании запасов.
    • Сотрудник расхода. В этом поле указывается сотрудник при списании запасов.
    • Специализация. В данном поле вводится специализация сотрудника. Выбирается из справочника "Специализации".
  • Комментарий. Произвольный комментарий к документу.

 

Данный документ может быть создан на основании документа "Инвентаризация запасов".
