Документ "Перемещение денежных средств"

Документ "Перемещение денежных средств" предназначен для осуществления перемещения денежных средств между кассами, счетами, лицевыми счетами пациентов. Переход к списку документов производится из раздела "Финансы" / Раздел "Денежные средства" / "Перемещения". 

 

В документе необходимо заполнить следующие реквизиты:

  • Сумма. Указывается сумма перемещения денежных средств.
  • Вид средств. Указывается вид денежных средств - наличные, безналичные, лицевой счет.
  • Статья движения. Указывается статья движения денежных средств.
  • Источник списания. Группа реквизитов:
    • Счет источника организации, для которого осуществляется перемещение. Становится доступным для указания при выбранном способе оплаты - безналичные. Выбирается из справочника "Банковские счета".
    • Касса источника организации, для которой осуществляется перемещение. Становится доступным для указания при выбранном способе оплаты - наличные. Выбирается из справочника "Кассы".
    • Пациент. Пациент, с лицевого счета которого осуществляется перемещение. Становится доступным для указания при выбранном способе оплаты - лицевой счет. Выбирается из справочника "Контрагенты".
    • Вид лицевого счета. Вид лицевого счета пациента с которого осуществляется перемещение. Становится доступным для указания при выбранном способе оплаты - лицевой счет. Выбирается из справочника "Виды лицевых счетов".
  • Приемник зачисления. Группа реквизитов:
    • Счет приемника организации, для которого осуществляется перемещение. Становится доступным для указания при выбранном способе оплаты - безналичные. Выбирается из справочника "Банковские счета".
    • Касса приемника организации, для которой осуществляется перемещение. Становится доступным для указания при выбранном способе оплаты - наличные. Выбирается из справочника "Кассы".
    • Пациент. Пациент, на лицевой счет которого осуществляется перемещение. Становится доступным для указания при выбранном способе оплаты - лицевой счет. Выбирается из справочника "Контрагенты".

    • Вид лицевого счета. Вид лицевого счета пациента на который осуществляется перемещение. Становится доступным для указания при выбранном способе оплаты - лицевой счет. Выбирается из справочника "Виды лицевых счетов".

  • Комментарий. Указывается комментарий к документу.
