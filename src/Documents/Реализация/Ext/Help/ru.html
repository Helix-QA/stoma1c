Документ "Реализация"

Документ "Реализация товаров" предназначен для оформления продажи запасов конкретному покупателю. Переход к списку документов производится из раздела "Оказание услуг" / Раздел "Продажи товаров" / "Реализации". 

В документе необходимо указать значения следующих полей:

  • Контрагент/Пациент. В этом поле необходимо указать покупателя, которому будет отгружен реализуемый запас. Значение выбирается из справочника "Контрагенты".
  • Склад. В этом поле необходимо указать склад организации из которого товар будет отгружаться покупателю.
  • Сотрудник. В этом поле необходимо указать сотрудника, ответственного за продажу.
  • Диск. карта. В этом поле указывается дисконтная карта пациента.

Документ имеет несколько табличных частей, которые располагаются на соответствующих закладках.
Информация о реализации запасов отображается на закладке "Запасы".

  • Запасы. Табличная часть:
    • Номенклатура. В данное поле вводится элемент справочника номенклатуры.
    • Характеристика, серия. Указывается характеристика номенклатурной позиции. Поле доступно только в том случае, если при настройке параметров учета указано ведение учета по партиям номенклатуры. Выбирается из справочника "Характеристики, серии".
    • Кол-во. Указывается количество единиц измерения номенклатурной позиции.

    • Упаковка. - упаковка номенклатуры, если есть. Выбирается из справочника "Упаковки номенклатуры".

    • Цена - цена реализуемой номенклатурной позиции.

    • % ручн. скидки - % ручной скидки по строке.
    • Сумма авт. скидки - сумма автоматической скидки по данной строке.

    • Сумма - стоимость номенклатуры по данной строке.

    • Ставка НДС - ставка налога на добавленную стоимость.

    • Сумма НДС - сумма налога на добавленную стоимость по данной строке.

    • Всего - итоговая стоимость номенклатуры по данной строке.

Информация о реализации услуг отображается на закладке "Услуги".

  • Услуги. Табличная часть:

    • Номенклатура. В данное поле вводится элемент справочника номенклатуры.
    • Кол-во. Указывается количество единиц измерения номенклатурной позиции.

    • Цена - цена реализуемой номенклатурной позиции.

    • % ручн. скидки - % ручной скидки по строке.
    • Сумма авт. скидки - сумма автоматической скидки по данной строке.

    • Сумма - стоимость номенклатуры по данной строке.

    • Ставка НДС - ставка налога на добавленную стоимость.

    • Сумма НДС - сумма налога на добавленную стоимость по данной строке.

    • Всего - итоговая стоимость номенклатуры по данной строке.

  • Комментарий. Произвольный комментарий к документу.

 

На основании данного документа могут быть оформлены следующие документы:

  • "Возврат от пациента";
  • "Поступление денежных средств";
  • "Списание денежных средств".
